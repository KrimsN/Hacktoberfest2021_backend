# Hacktoberfest2021_backend

## [heroku app](https://hacktoberfest2021-backend.herokuapp.com/docs)

## how to install:

dependency:

- python 3.9
- pipenv

you need to run: 

`pipenv install`

to installation requirements

setup env vars:

- PYTHONPATH = .
- SECRET_KEY = YourSecretKey

and to run:

`pipenv run python app/main.py`
